# Spacing System
COVE provides a basic, custom spacing system for quickly applying padding or margins to elements.

## Available Classes & Values
The following classes and position values are available:

**Padding**
- `.p-`: Padding for all sides of the element.
- `.pt-`: Padding for the top of the element.
- `.pr-`: Padding for the right of the element.
- `.pb-`: Padding for the bottom of the element.
- `.pl-`: Padding for the left of the element.
- `.px-`: Padding for the left and right of the element.
- `.py-`: Padding for the top and bottom of the element.

<br/>

**Margin**
- `.m-`: Margin for all sides of the element.
- `.mt-`: Margin for the top of the element.
- `.mr-`: Margin for the right of the element.
- `.mb-`: Margin for the bottom of the element.
- `.ml-`: Margin for the left of the element.
- `.mx-`: Margin for the left and right of the element.
- `.my-`: Margin for the top and bottom of the element.

<br/>

**Size Values**

The size values are based on 8px increments per 1 unit. The following values are available:
- `auto`: auto (margin only)

- `0`: 0px
- `1`: 8px
- `2`: 16px
- `3`: 24px
- `4`: 32px
- `5`: 40px
- `6`: 48px
- `7`: 56px
- `8`: 64px
