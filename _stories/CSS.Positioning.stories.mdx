# Grid System
COVE provides a custom classname positioning system for justifying and aligning content, items, or self.

## Syntax
```css
.cove-justify-{property}-{position-value} {
  justify-{property}: {value};
}

.cove-align-{property}-{position-value} {
  align-{property}: {value};
}
```

## Available Classes & Values
The following classes and position values are available:

**Justify**
- `.cove-justify-content-`<div class="classColor">\{position-value\}</div>
- `.cove-justify-items-`<div class="classColor">\{position-value\}</div>
- `.cove-justify-self-`<div class="classColor">\{position-value\}</div>

**Align**
- `.cove-align-content-`<div class="classColor">\{position-value\}</div>
- `.cove-align-items-`<div class="classColor">\{position-value\}</div>
- `.cove-align-self-`<div class="classColor">\{position-value\}</div>

**Position Values**
- `flex-start`
- `flex-end`
- `center`
- `baseline`
- `stretch`
- `space-between`
- `space-around`
