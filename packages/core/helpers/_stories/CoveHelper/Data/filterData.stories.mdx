# filterData
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that filters data based on the values defined in `filters`.

&nbsp;
## Syntax
```js
CoveHelper.Data.filterData(filters, data)
```

## Parameters
This function takes in two arguments: `filters` and `data`:
- `filters`: Array of objects where each object contains two properties: `columnName` and `active`:
  - `columnName`: String representing the name of a column in data.
  - `active`: Value indicating whether the rows in data with that column value should be included in the filtered result.
- `data`: Array of objects where each object represents a row of data.
<br/>

**Returns:** New array containing only the rows that meet the filter criteria.

&nbsp;
## Example
```jsx
const personFilters = [
  { columnName: 'name', active: 'Alice' },
  { columnName: 'age', active: 30 }
]

const personData = [
  { name: 'Alice', age: 30, height: 5.6, weight: 130 },
  { name: 'Bob', age: 40, height: 5.9, weight: 160 },
  { name: 'Charlie', age: 50, height: 5.11, weight: 180 }
]

const filteredPerson = CoveHelper.Data.filterData(personFilters, personData)
console.log(filteredPerson) // [{ name: 'Alice', age: 30, height: 5.6, weight: 130 }]
```
