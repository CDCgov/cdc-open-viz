# getFilteredDataColumns
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that iterates over a provided array of data and returns an array of filtered column names that are in the provided config object.

&nbsp;
## Syntax

```js
CoveHelper.Data.getFilteredDataColumns(config, dataArray)
```

## Parameters
- `config`: Object that may contain two keys: `series` and `confidenceKeys`:
  - `series`: Array of objects that each contain a dataKey property.
  - `confidenceKeys`: Object that contains key-value pairs where the keys are column names and the values are boolean values indicating whether the column is a confidence interval.
- `dataArray`: Array of objects where each object represents a row of data.
<br/>

**Returns:** An array of filtered data column names from `dataArray`.

&nbsp;
## Example
```js
const config = {
  series: [
    { dataKey: 'name' },
    { dataKey: 'age' }
  ],
  confidenceKeys: {
    height: true,
    weight: true
  }
}

const dataArray = [
  { name: 'Alice', age: 30, height: 5.6, weight: 130 },
  { name: 'Bob', age: 40, height: 5.9, weight: 160 },
  { name: 'Charlie', age: 50, height: 5.11, weight: 180 }
]

const columns = CoveHelper.Data.getFilteredDataColumns(config, dataArray)
console.log(columns) // Output: ['height', 'weight']
```
