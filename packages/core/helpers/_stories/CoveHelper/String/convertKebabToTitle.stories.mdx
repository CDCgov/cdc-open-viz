# convertKebabToTitle
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that takes a kebab-case string and converts it to a title case string with spaces between each word.

&nbsp;
## Example
```js
CoveHelper.String.convertKebabToTitle("foo-bar-baz"); // "Foo Bar Baz"
```
