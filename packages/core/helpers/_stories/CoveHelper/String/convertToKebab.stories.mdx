# convertToKebab
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that takes a string and converts it to kebab-case.

&nbsp;
## Example
```js
CoveHelper.String.convertToKebab("fooBarBaz"); // "foo-bar-baz"
```
