# convertToPascal
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that takes a string in kebab-case or snake_case and converts it to PascalCase.

&nbsp;
## Example
```js
CoveHelper.String.convertToPascal("foo-bar-baz"); // "FooBarBaz"

CoveHelper.String.convertToPascal("foo_bar_baz"); // "FooBarBaz"
```
