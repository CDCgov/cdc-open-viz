# convertPascalToCamel
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that takes a PascalCase string and converts it to a camelCase string.

&nbsp;
## Example
```js
CoveHelper.String.convertPascalToCamel("FooBarBaz"); // "fooBarBaz"
```
