# capitalizeFirstLetter
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that takes a string and capitalizes the first letter.

&nbsp;
## Example
```js
CoveHelper.String.capitalizeFirstLetter("foo"); // "Foo"
```
