# convertToCamel
This function takes a string in kebab-case or snake_case and converts it to camelCase.

&nbsp;
### Example
```js
CoveHelper.String.convertToCamel("foo-bar-baz"); // "fooBarBaz"
CoveHelper.String.convertToCamel("foo_bar_baz"); // "fooBarBaz"
```
