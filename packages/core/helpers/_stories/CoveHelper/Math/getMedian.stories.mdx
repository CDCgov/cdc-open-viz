# getMedian
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that takes an array of numbers and returns the median of all its elements. It first calculates the midpoint index of the array,
sorts the array in ascending order, and returns the middle element for odd-length arrays, or the average of the middle two elements for even-length arrays.

&nbsp;
## Syntax
```js
CoveHelper.Math.getMedian(arr)
```

## Parameters
- `arr`: Array of numbers to be used.
<br/>

**Returns:** The median of the array, or `undefined` if the array is empty or contains non-numbers.

&nbsp;
## Example
```js
CoveHelper.Math.getMedian([1, 2, 3]); // 2

CoveHelper.Math.getMedian([1, 2, 3, 4]); // 2.5

CoveHelper.Math.getMedian("1,2,3"); // undefined
```
