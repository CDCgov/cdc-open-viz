# getSum
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that takes an array of numbers and returns the sum of all its elements. If the input is valid, the function maps the array elements to numbers, sums them up using the reduce() method, and returns the result.

&nbsp;
## Syntax
```js
CoveHelper.Math.getSum(arr);
```

## Parameters
- `arr`: Array of numbers to be summed.
<br/>

**Returns:** The sum of all the elements in the array.

&nbsp;
## Example
```js
CoveHelper.Math.getSum([1, 2, 3]); // 6

CoveHelper.Math.getSum("1,2,3"); // undefined
```
