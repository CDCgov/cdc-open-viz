# getMean
This function takes an array of numbers and returns the mean (average) of all its elements. If the input is valid, the function
maps the array elements to numbers, removes any zero values if the ignoreZeros parameter is true, calculates the mean
using the reduce() method, and returns the result.

&nbsp;
## Syntax
```js
CoveHelper.Math.getMean(arr, ignoreZeros)
```

**Parameters**:

- `arr`: The array of numbers to be averaged.
- `ignoreZeros`: Whether to ignore zero values when calculating the mean. Default is `false`.

&nbsp;
### Example
```js
CoveHelper.Math.getMean([1, 2, 3]); // returns 2
CoveHelper.Math.getMean([1, 0, 3], true); // returns 2
CoveHelper.Math.getMean("1,2,3"); // returns undefined
```
