# getMean
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that takes an array of numbers and returns the mean (average) of all its elements. If the input is valid, the function
maps the array elements to numbers, calculates the mean
using the reduce() method, and returns the result.

If `ignoreZeros` is set to `true`, removes any `0` values in the provided array from the calculation.

&nbsp;
## Syntax
```js
CoveHelper.Math.getMean(arr, ignoreZeros)
```

## Parameters
- `arr`: Array of numbers to be averaged.
- `ignoreZeros`: Boolean designating whether to ignore zero values when calculating the mean. Default is `false`.
<br/>

**Returns:** The mean (average) of the provided array elements, or `undefined` if the input is invalid.

&nbsp;
## Example
```js
CoveHelper.Math.getMean([1, 2, 3]); // 2

CoveHelper.Math.getMean([1, 0, 3], true); // 2

CoveHelper.Math.getMean("1,2,3"); // undefined
```
