# getMode
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that takes an array of numbers and returns an array containing the mode(s) of its elements. The mode is the number(s) that appears most frequently in the input array.

&nbsp;
## Syntax
```js
CoveHelper.Math.getMode(arr)
```

## Parameters
- `arr`: Array of numbers to be used.
<br/>

**Returns:** An array containing the mode(s) of the input array.

&nbsp;
## Example
```js
CoveHelper.Math.getMode([1, 2, 2, 3, 3, 3]) // [3]

CoveHelper.Math.getMode([1, 2, 2, 3, 3, 3, 4, 4]) // [3]
```
