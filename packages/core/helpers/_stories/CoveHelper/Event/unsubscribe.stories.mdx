# unsubscribe
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that removes an event listener for a given event name from the document object. The listener function to be removed should be
the same as the one added previously using the [subscribe](/?path=/docs/helpers-covehelper-event-subscribe--docs) function.

&nbsp;
## Syntax
```js
CoveHelper.Event.unsubscribe(eventName, listener);
```

## Parameters
- `eventName`: String representing the name of the event to remove the listener from.
- `listener`: Function to be removed from the event listener list.
<br/>

**Returns:** The result of the `removeEventListener` method call.

&nbsp;
## Example
In this example, the `handleClick` function is registered as an event listener for the `click` event
on a button element using the `subscribe` function. The same function is later removed as a listener using the `unsubscribe` function.
```js
function handleClick(event) {
  console.log('Button clicked!', event.target);
}

const button = document.querySelector('button');
subscribe('click', handleClick);

// Later, to remove the listener:
unsubscribe('click', handleClick);
```
