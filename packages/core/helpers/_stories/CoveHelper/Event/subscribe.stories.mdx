# subscribe
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that registers an event listener for a given event name on the document object. When the specified event is triggered,
the listener function will be executed.

See the [unsubscribe](/?path=/docs/helpers-covehelper-event-unsubscribe--docs) function for removing the event listener.

&nbsp;
## Syntax
```js
CoveHelper.Event.subscribe(eventName, listener);
```

## Parameters
- `eventName`: String representing the name of the event to listen for.
- `listener`: Function to be executed when the specified event is triggered.
<br/>

**Returns:** The result of the `addEventListener` method call.

&nbsp;
## Example
In this example, the `handleClick` function is registered as an event listener for the click event on a button element using the `subscribe` function.
```js
function handleClick(event) {
  console.log('Button clicked!', event.target);
}

const button = document.querySelector('button');
subscribe('click', handleClick);
```
