# parseDate
A [CoveHelper](./?path=/docs/helpers-covehelper-covehelper-class--docs) function that uses the d3-time-format library to parse a given date string according to a specified format.

&nbsp;
## Syntax
```js
CoveHelper.Date.parseDate(format, dateString)
```

## Parameters
- `format`: String representing the format of the input date string.
  - If not provided, it defaults to **%Y-%m-%d**, which represents the year, month, and day of the date in a numeric format, separated by hyphens.
- `dateString`: Date string to be parsed.
<br/>

**Returns:** The function returns a Date object representing the parsed date string. If the parsing fails, it returns the current date.

&nbsp;
## Example
```jsx
const dateString = '01-Jan-2022';

const parsedDate = CoveHelper.Date.parseDate('%d-%b-%Y', dateString);

console.log(parsedDate); // Sat Jan 01 2022 00:00:00 GMT+0000 (Coordinated Universal Time)
```
