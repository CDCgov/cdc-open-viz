# DataTransform
The DataTramsform class contains two methods for standardizing data; these standardizations ensure data is properly formatted for visualizations.

&nbsp;
## Constructor
The DataTransform class has a constructor method which sets the following constants:
- `errorMessageEmtpyData`: Error message for empty data
- `errorMessageFormat`: Error message for unsupported datatype
- `descriptionHeader`: Constant for description header
- `descriptionRoot`: Constant for description root
- `descriptionSeriesFilter`: Constant for description series filter

&nbsp;
## Available Methods
* [autoStandardize](./?path=/docs/helpers-datatransform-autostandardize--docs) - Performs baseline standardizations.
* [developerStandardize](./?path=/docs/helpers-datatransform-developerstandardize--docs) - Performs standardizations based on provided descriptions of the data.
