# dataSlice
The **dataSlice** of the COVE store contains the state for data. This includes visualization data, datasets, filtered data, and formatted data.

It also provides a set list of functions that can be used to modify the `data` state:

&nbsp;
## Structure
The `dataSlice` structure is as follows:

```json
{
  "data": [],
  "datasets": [],
  "filteredData": [],
  "formattedData": [],
}
```

- **data**: An array of objects containing the data for the visualization.
- **datasets**: An array of objects containing multiple sets of data.
- **filteredData**: An array of objects containing filtered data for a visualization.
- **formattedData**: An array of the data that has been formatted for use with the visualization.

&nbsp;
## Functions
<br/>
### setData()
Sets the state's `data` property to the provided value.

**Syntax**
```javascript
setData(data)
```
