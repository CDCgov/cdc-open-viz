# Tooltip Component
The `<Tooltip>` component is used to display a tooltip when hovering over a specified element. It should normally only be used for one-off tooltip displays.

The inline `tooltip` method for components should be used instead, when available. Most [input components](./?path=/docs/components-input-text--docs) provide `tooltip` - and tooltip related - properties, which automatically handle tooltip setup for you.
