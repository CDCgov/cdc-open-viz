The Icon component allows you to reference, and insert, a pre-defined svg image anywhere in your code. A full list of available icons can be found below.

## Display Options
By default, the Icon component displays the svg inside a styled span which inherits your current font color and size.

```html
<span class="cove-icon">
    <svg ... />
</span>
```

If you prefer to use just the base svg - without the wrapping span - simply add the `base` property to the component.
```jsx
  <Icon display="iconHash" base />
```
