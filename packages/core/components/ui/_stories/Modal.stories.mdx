## Usage Examples
Modals can be customized to fix various display requirements, depending on the situation. Below are a few examples highlighting various potential use cases.

### Interaction
<Canvas>
  <Story name="Interaction">
    {() => {
      /* Mockup overlay context data for demo. Use Overlay component methods for live version. */
      const [ showModal, setShowModal ] = useState(false)
      let data = {
        show: showModal,
        actions: {
          toggleOverlay: (display = false) => {
            setShowModal(display)
          }
        }
      }
      /* ------------------------------------------------------------------------------------ */
      return (
        <>
          <Button onClick={() => data.actions.toggleOverlay(true)}>Trigger Modal Open</Button>
          <Overlay override={data}>
            <Modal fontTheme={'light'} headerBgColor={'#824c88'} override={data}>
              <Modal.Header>
                Example Modal
              </Modal.Header>
              <Modal.Content>
                <p>This is a popup modal with general information. This serves as an example of how interaction with the modal is intended to work.</p>
              </Modal.Content>
            </Modal>
          </Overlay>
        </>
      )
    }}
  </Story>
</Canvas>
